# Проверка на доступ к ресурсам пользователя по роли, статусу и возрасту

Консольное приложение для проверки по заданным параметрам.


## Функциональность

Программа позволяет:
* ввести возраст, роль и статус;
* провести определенные проверки и вывести доступность в формате "true" или "false";


## Критерии

| Диапазон ИМТ                   | Категория                                              |
|--------------------------------|--------------------------------------------------------|
| возраст $< 18$ лет             | Доступ запрещен                                        |
| роль "admin" или "moderator"   | Доступ разрешен в любом случае, независимо от возраста |
| статус "inactive" или "paused" | Доступ запрещен                                        |
